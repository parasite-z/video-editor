# 🎬 VIDEO EDITOR — คู่มือติดตั้ง (ใช้เวลาประมาณ 5 นาที ไม่ต้องเขียนโค้ด)

ตัวนี้จะเพิ่ม "นักตัดต่อวิดีโอ" เข้าไปใน Claude Code พอติดตั้งเสร็จแล้ว คุณแค่พิมพ์สั่งเป็นภาษาคนปกติ เช่น:

- "ตัด 2 วินาทีแรกออก"
- "ทำให้เป็นแนวตั้งสำหรับ TikTok"
- "ใส่หัวเรื่องว่า สินค้าใหม่"
- "ขอดูตัวอย่างหน่อย"
- "เรนเดอร์ออกมาเลย"

---

## ⚙️ ก่อนเริ่ม — ต้องมี 2 แอปนี้ก่อน (ฟรีทั้งคู่)

1. **Claude Code** — ทีมของคุณน่าจะมีอยู่แล้ว
2. **Node.js** — เข้า https://nodejs.org แล้วกดปุ่มใหญ่ที่เขียนว่า **"LTS"**

> ตัวติดตั้งด้านล่างจะเช็กให้เองว่ามีครบไหม ถ้าขาดอะไรมันจะเปิดหน้าดาวน์โหลดให้ แค่ติดตั้งตามที่มันบอก แล้วรันใหม่อีกครั้ง

---

## 💻 วิธีติดตั้ง

> ⚠️ **แตกไฟล์ (unzip) ก่อนเสมอ** อย่าเปิดจากในไฟล์ .zip โดยตรง

### บนเครื่อง Mac

ลองวิธีแรกก่อน: **ดับเบิลคลิก** ไฟล์ `Install-Mac.command`

**ถ้าเปิดไม่ได้** — ไฟล์ไปเปิดใน TextEdit, หรือขึ้นข้อความ *"unidentified developer" / "cannot be opened"*:

นี่เป็นเรื่องปกติของไฟล์ที่ดาวน์โหลดมา ให้ใช้วิธีนี้แทน (ได้ผลแน่นอน 100%):

1. กด **Cmd + Space** → พิมพ์ **Terminal** → กด Enter
   (จะมีหน้าต่างสีดำชื่อ "Terminal" เปิดขึ้นมา)
2. ในหน้าต่างนั้น พิมพ์คำว่า **`bash`** แล้วเคาะ **เว้นวรรค 1 ที** — **ยังไม่ต้องกด Enter**
3. **ลาก** ไฟล์ `Install-Mac.command` จาก Finder มา **วางลงบนหน้าต่าง Terminal**
   (ที่อยู่ของไฟล์จะถูกเติมให้เองอัตโนมัติ)
4. กด **Enter**

จากนั้นทำตามข้อความในหน้าต่าง พอขึ้นคำว่า **"Done!"** ก็เสร็จเรียบร้อย

### บนเครื่อง Windows
1. **ดับเบิลคลิก** ไฟล์: `Install-Windows.bat`
2. ถ้า Windows ขึ้นกล่องสีน้ำเงิน *"Windows protected your PC"*:
   - กด **"More info"** → **"Run anyway"**
3. ทำตามข้อความในหน้าต่าง พอขึ้นคำว่า **"Done!"** กดปุ่มใด ๆ เพื่อปิด

## ▶️ วิธีใช้งาน (หลังติดตั้งเสร็จ)

1. เปิด **Claude Code**

2. สร้างโปรเจกต์วิดีโอขึ้นมาก่อน (ทำครั้งแรกครั้งเดียว หรือทุกครั้งที่จะเริ่มวิดีโอใหม่) พิมพ์:

   ```
   npx hyperframes init my-video --example blank --non-interactive
   ```

3. เปิดโฟลเดอร์ `my-video` ใน Claude Code แล้วพิมพ์:

   ```
   /video-editor
   ```

   จากนั้นสั่งงานด้วยคำพูดของคุณเองได้เลย:

   - ใส่หัวเรื่องว่า ยินดีต้อนรับ
   - ทำให้ช่วงเปิดสั้นลง
   - เพิ่มเสียงพากย์พูดว่า "พบกับเลนส์รุ่นใหม่"
   - ทำให้เป็นแนวตั้งสำหรับ TikTok
   - ขอดูตัวอย่าง
   - เรนเดอร์ออกมา

---

## ❓ มีปัญหา?

- **รันตัวติดตั้งซ้ำได้** — ปลอดภัย รันกี่ครั้งก็ได้
- ต้อง **แตกไฟล์ (unzip) ก่อน** อย่ารันจากในไฟล์ .zip โดยตรง
- การดูตัวอย่าง/เรนเดอร์ต้องมี **FFmpeg** — ถ้ายังไม่มี ตัวติดตั้งจะบอกวิธีลง
- ยังติดอยู่? ถ่ายภาพหน้าจอหน้าต่างตัวติดตั้ง ส่งให้คนที่แชร์ไฟล์นี้มาให้คุณ
